What is Library?
- Library is a set of reusable functions used by computer programs.
- You are in full control when you call a method from a library and the control is then returned.
- It's incorporated seamlessly into existing projects to add functionality that you can access using an API.
- They are important in program linking and binding process.
Example: jQuery is a JavaScript library that simplities DOM manipulation.

What is Framework?
- Framework is a piece of code that dictates the architecture of your project and aids in programs.
- The code never calls into aframework, instead the framework calls you.
- It cannot be seamlessly incorporated into an existing project. Instead it can be used when a new project is started.
- They provide a standard way to build and deploy applications
Example: AngularJS is a JavaScript-based framework for dynamic web applications.

Flask Quickstart Documents:
https://flask.palletsprojects.com/en/stable/quickstart/

A minimal Flask application looks something like this:

from flask import Flask

app = Flask(__name__)

@app.route("/")
def hello_world():
    return "<p>Hello, World!</p>"

To run the flask server: use this command in terminal
-> flask --app hello_flask run. # hello_flask is filename

OR 
-> export FLASK_APP=hello_flask.py 
-> flask run
Now it will give uou the localhost url to access the server.

To stop the flask:
-> jobs  # it will show the running jobs
-> kill %2  # 2 is the job id
